## Portal de Notícias do CACIC no Ar

O CACIC lança hoje seu portal de notícias oficial. A partir de agora, todas as informações relevantes para os estudantes de Ciências da Computação da UTFPR Santa Helena serão publicadas aqui.

## O que Você Vai Encontrar

Nesta seção de notícias, publicaremos:

- Avisos sobre eventos acadêmicos e culturais
- Resultados de eleições e assembleias do CACIC
- Parcerias com empresas e organizações externas
- Oportunidades de estágio e emprego na área de TI
- Comunicados importantes da coordenação do curso

## Agenda de Eventos 2026

Confira os principais eventos previstos para este semestre:

1. Semana Acadêmica de Computação (SECOMP) — Abril 2026
2. Hackathon Interno CACIC — Maio 2026
3. Visita técnica a empresas de tecnologia — Junho 2026
4. Mostra de TCCs — Julho 2026

## Como Receber as Novidades

Para se manter atualizado, acesse este portal regularmente. Em breve teremos:

```typescript
// Futuramente: API de notificações
const subscribe = async (email: string) => {
  await fetch('/api/notificacoes', {
    method: 'POST',
    body: JSON.stringify({ email }),
  })
}
```

Use o token `text.primary` do nosso design system para garantir legibilidade em qualquer tema.
