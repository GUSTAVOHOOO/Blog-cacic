## Bem-vindos ao Blog Oficial do CACIC

O Centro Acadêmico de Ciências da Computação da UTFPR Santa Helena estreia seu blog oficial — um espaço dedicado a estudantes, professores e entusiastas da computação.

Aqui você encontrará artigos técnicos, notícias sobre eventos acadêmicos, trabalhos de conclusão de curso e muito mais. Nossa missão é ser o ponto de referência digital dos estudantes de CC da UTFPR-SH.

## O que Publicaremos

Nosso conteúdo será organizado em categorias:

- **Institucional** — notícias e comunicados do CACIC
- **Técnico** — tutoriais, artigos e descobertas em computação
- **Acadêmico** — TCCs, trabalhos e pesquisas dos estudantes
- **Eventos** — hackathons, palestras e semanas acadêmicas

## Tecnologias do Blog

Este blog foi construído com tecnologias modernas:

1. Next.js 14 com App Router
2. TypeScript para tipagem estática
3. Chakra UI v3 para o design system
4. Supabase para banco de dados e autenticação
5. MDX para conteúdo rico com componentes interativos

## Um Exemplo de Código

Veja como é simples criar um componente React com TypeScript:

```typescript
interface Props {
  title: string
  author: string
  date: string
}

export function PostHeader({ title, author, date }: Props) {
  return (
    <header>
      <h1>{title}</h1>
      <p>Por {author} — {date}</p>
    </header>
  )
}
```

Para estilizar, usamos o token `bg.canvas` do Chakra UI que respeita automaticamente o tema claro e escuro.
