**src/lib/validations/contato.ts:**
```typescript
import { z } from 'zod'

export const contatoSchema = z.object({
  nome: z.string().min(2, 'Nome deve ter pelo menos 2 caracteres').max(100, 'Nome muito longo'),
  email: z.string().email('E-mail inválido'),
  assunto: z.enum(['Dúvida', 'Sugestão', 'Parceria', 'Outro'], {
    errorMap: () => ({ message: 'Selecione um assunto válido' }),
  }),
  mensagem: z.string().min(10, 'Mensagem deve ter pelo menos 10 caracteres').max(2000, 'Mensagem muito longa'),
})

export type ContatoInput = z.infer<typeof contatoSchema>
```

**src/lib/validations/upload.ts:**
Export constants:
- `ALLOWED_IMAGE_TYPES = ['image/jpeg', 'image/png', 'image/webp'] as const`
- `ALLOWED_DOC_TYPES = ['application/pdf'] as const`
- `MAX_IMAGE_SIZE = 2 * 1024 * 1024` (2MB in bytes)
- `MAX_DOC_SIZE = 10 * 1024 * 1024` (10MB in bytes)

Export `validateUpload(file: { type: string; size: number; name: string }): { valid: true; category: 'image' | 'document' } | { valid: false; error: string }`:
- Check if type is in ALLOWED_IMAGE_TYPES → category 'image', check size <= MAX_IMAGE_SIZE
- Check if type is in ALLOWED_DOC_TYPES → category 'document', check size <= MAX_DOC_SIZE
- If type not in either → return { valid: false, error: `Tipo de arquivo não permitido. Permitidos: JPEG, PNG, WebP, PDF` }
- If image too large → return { valid: false, error: 'Imagem deve ter no máximo 2MB' }
- If PDF too large → return { valid: false, error: 'PDF deve ter no máximo 10MB' }
- If valid → return { valid: true, category }

Export a `rateLimiter` factory function that creates pre-configured limiters:
```typescript
export function rateLimiter(options: {
  requests: number
  window: `${number} s` | `${number} m` | `${number} h`
  prefix: string
}): (identifier: string) => Promise<RateLimitResult>
```

Implementation: create a `new Ratelimit` with `slidingWindow(options.requests, options.window)` algorithm and `prefix: options.prefix`. Return an async function that calls `ratelimit.limit(identifier)` and returns the result shaped as RateLimitResult.
